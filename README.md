1. Follow the instructions in `Environment Setup Guide.pdf`

## Install Instructions
1. Clone repository
```
git clone https://github.com/vch2128/2024_ICG_HW4
```
2. Re-install glm
```
cd 2024_ICG_HW4
rmdir glm
git clone https://github.com/g-truc/glm.git
```
3. Re-build the project
```
cd ..
rm -rf build
mkdir build
cd build
cmake ..
```
